## Project Experience

### Hand-Gesture-Recognition System Port for a Novatek Development Board

_First-year team project · Primary technical implementer · [NCKU CSIE Project Exhibition](https://news-secr.ncku.edu.tw/p/405-1037-298426,c5934.php?Lang=zh-tw)_

I was responsible for the end-to-end technical implementation of porting a MediaPipe Hands-style two-stage hand-tracking pipeline to the Novatek NT9869x embedded platform. The port preserved the graph, geometry, and post-processing flow while replacing the original inference path with the board's on-chip NPU backend.

- Ported the palm-detection and hand-landmark pipeline to the NT9869x, replacing the original inference calculator path with a custom NPU-backed inference calculator.
- Integrated the live camera → ISP / VPE → MediaPipe graph → NPU → HDMI overlay data path.
- Implemented the Cortex-A73 host-side flow for graph execution, SSD decode, NMS, rotated ROI handling, landmark projection, gesture rules, result caching, and overlay rendering.
- Ran palm detection and 21-point hand-landmark inference on the on-chip NPU.
- Reworked image preparation to avoid full-frame RGB conversion by fusing NV12 conversion with inference-region crop and resize operations.
- Used SIMD / NEON-accelerated sampling for the rotated hand crop and image conversion path.
- Built the model-conversion and validation workflow for INT16 palm and landmark models, including host-side golden checks and a 12-frame review set.
- Implemented live skeleton, ROI, handedness, gesture-state, and timing-HUD overlays, together with five rule-based gesture classes.
- Prepared the project demonstration, performance measurements, technical explanation, and exhibition poster.

**Measured results:** stable **30 fps** live HDMI output; per-tick compute reduced from approximately **55 ms to 10 ms**; NV12-to-RGB processing reduced from **43.5 ms to 1.4 ms**. All 12 review frames passed the 0.90 validation threshold, with an average palm-detector score of **0.949** and average hand-landmark presence score of **0.977**.

**Stack:** MediaPipe 0.10.35, Novatek NT9869x, HDAL, Zig, C++20, Python, SIMD / NEON

_Public poster available as a [PDF](/files/nt9869x-hand-tracking-poster.pdf). The implementation repository and internal project materials are private._
